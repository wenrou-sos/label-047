## 1. 产品概述

小区物业管理面板系统是一个面向物业管理人员的综合性管理平台，旨在实现物业费缴纳管理、维修报修跟踪和数据可视化分析的一体化解决方案。目标用户为小区物业管理人员和维修调度人员，核心价值在于提升物业费收缴效率、优化报修响应速度、通过数据驱动决策。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 物业管理员 | 系统分配 | 物业费管理、报修分配、数据查看 |
| 维修人员 | 系统分配 | 接单、更新报修进度、上传维修结果 |

### 2.2 功能模块

1. **仪表盘首页**：关键指标总览、物业费收缴率、报修处理率、近期动态
2. **物业费管理页**：住户缴费状态查询、欠费提醒、多维筛选
3. **报修管理页**：报修单列表、进度跟踪、维修人员分配、实时位置追踪
4. **数据分析页**：物业费统计图表、报修趋势分析、核心指标仪表盘

## 3. 核心流程

**物业费管理流程**：管理员登录 → 查看欠费概览 → 按条件筛选住户 → 查看欠费详情 → 发送催缴提醒 → 标记缴费状态

**报修管理流程**：住户报修 → 管理员受理 → 分配维修人员 → 维修人员出发（实时位置追踪）→ 到达处理 → 上传维修结果 → 关闭工单

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色系背景（#0f172a 深蓝黑），搭配青蓝色强调色（#06b6d4 cyan-500），辅助色为翡翠绿（#10b981 emerald-500）表示成功/已缴费，琥珀色（#f59e0b amber-500）表示警告/欠费，玫红色（#f43f5e rose-500）表示紧急/逾期
- **按钮风格**：圆角8px，微阴影，hover时亮度提升和缩放效果
- **字体**：数字使用等宽字体（JetBrains Mono），中文使用系统默认，标题使用大号粗体
- **布局风格**：左侧固定导航栏 + 右侧内容区，卡片式内容组织
- **图标风格**：Lucide线性图标，统一24px尺寸
- **数据可视化**：ECharts图表使用与主题一致的配色方案

### 4.3 响应式设计

- 桌面端（≥1280px）：完整侧边栏 + 内容区三栏布局
- 平板端（768px-1279px）：折叠侧边栏为图标模式 + 内容区双栏布局
- 移动端（<768px）：底部导航栏 + 单栏堆叠布局，表格改为卡片展示
